# SDK drop-in folder

Replace these five files with whatever the SDK team ships:

- `rust_wallet_sdk.js`
- `rust_wallet_sdk_bg.js`
- `rust_wallet_sdk_bg.wasm`
- `rust_wallet_sdk.d.ts`
- `rust_wallet_sdk_bg.wasm.d.ts`

Then refresh the browser — Vite picks up the new bytes automatically.

Do **not** delete this README. Everything else in this folder is fair game to
overwrite.
